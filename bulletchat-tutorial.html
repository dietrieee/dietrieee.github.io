How to Set Up BulletChat for YouTube LiveChat

  1. Download and install the BulletChat app. (yournameChat.msi)
  2. Download and install Mica For Everyone. here
  3. Open Mica For Everyone and click on Add New Rule > Add Process Rule.
  4. Search for the app name: BulletChat (yournameChat) -> You need to open the BulletChat App first.
  5. Set the backdrop type to Acrylic and enable Blur Behind.
  6. Ensure Transparency Effects are enabled in your Windows Settings:
    Go to Settings > Personalization > Colors and turn on Transparency Effects.
  7. Run server.exe included with the app.
  8. Copy your YouTube Livestream ID (youtube.com/watch?v=here_is_the_id) and paste it into the BulletChat app.
  9. Click the Connect button.
  10. In your streaming software (e.g., OBS), add a Window Capture source and set the Capture Method to Windows 10.
  11. You're done! BulletChat should now display your YouTube LiveChat.
